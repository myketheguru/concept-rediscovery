# How this fits the program

One research program — **the powers and limits of representation** — on three tracks:

- **Track A — Foundations** (search is conserved; structured collapse under the matched prior; the
  representation-independent floor; the boundary). → [representation-dial](https://github.com/myketheguru/representation-dial), folders 01–03.
- **Track B — Concept-rediscovery** *(this repo)*. The original experiment: can a compression
  objective rediscover named human concepts from exact-label data? The most differentiated work of
  the program — and the most honest, because the headline hypothesis nulled.
- **Track C — The frontier: the access/search dial** (does matched geometry convert search into cheap
  access?). → [representation-dial](https://github.com/myketheguru/representation-dial), folder 04.

The through-line: **a representation/prior buys tractability and fidelity, not new capability —
capability tracks the alignment between the prior and the structure that's actually there.** Track A
shows it in theory, Track B tests it on data (and finds the prior load-bearing), Track C asks how far
a *matched* representation can push the access/search boundary.
